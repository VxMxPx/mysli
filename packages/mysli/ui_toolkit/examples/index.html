Mysli UI Toolkit :: Examples

Introduction

This is an example of UI Toolkit. To use it properly run PHP build-in server, in the root of this package's directory (ui_toolkit):

php -S localhost:8000

Then open the following url: http://localhost:8000/examples

Please note: this Toolkit is used above all, for mysli/dashboard and components related to it. This is alpha version, still under heavy development.

It's not recommended that you're use this toolkit on frontend.
